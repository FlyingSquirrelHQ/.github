# 🐿️ FlyingSquirrelOps

---

## 🌲 What is FlyingSquirrelOps?

FlyingSquirrelOps is a home for infrastructure patterns, operational playbooks, and tooling designed to make complex systems easier to run, maintain, and trust.

Built from real production experience—focused on clarity, repeatability, and long-term sustainability.

---

## 🧰 What you'll find here

* **Playbooks**
  Backup, restore, and migration guides used in real environments

* **Infrastructure as Code**
  Terraform and Kubernetes patterns for consistent deployments

* **Operational tooling**
  Scripts and workflows to reduce manual effort and risk

---

## 🧭 Principles

* **Predictable** — systems behave as expected
* **Documented** — knowledge is shared, not tribal
* **Repeatable** — works across environments
* **Maintainable** — built for the long term
